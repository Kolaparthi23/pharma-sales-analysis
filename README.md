# Pharma Sales Analysis (SQL)

## Project Overview
This project analyzes pharmaceutical sales data using SQL to understand
sales performance, top contributors, channel distribution, and sales team effectiveness.

## Objectives
- Sales overview by distributor and product
- Top 5 products, customers, and cities by sales
- Sales split by channel and sub-channel
- Sales analysis by sales team and product/product class
- Identification of data quality issues
- Top sales team, sales manager analysis

## Data Exploration & Cleaning
- Reviewed dataset structure
- Identified null values
- Identified duplicate records (not removed)
- Identified negative sales values (returns)

## Analysis Performed
- Distributor and product sales overview
- Top products, customers, and cities
- Channel and sub-channel sales split
- Sales team and sales rep performance analysis

## Tools Used
- SQL
- Pandas (EDA)
- GitHub

## Notes
Negative sales values were treated as returns
